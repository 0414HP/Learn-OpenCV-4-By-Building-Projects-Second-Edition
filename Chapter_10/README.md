# Chapter 10. Developing Segmentation Algorithms for Text Recognition 

## Requirements

- OpenCV 4.0
- Tesseract

## Steps to build

To compile on linux, Mac or windows using Mingw

```
mkdir build
cd build
cmake ..
make
```

## Executables

The following applications are generated.

```
./segment  
./segmentOcr  
./segmentOcrHigh
```
